How to use?

Step 1: Put "Zas'Admin" ServerScript under "ServerScriptService"

Step 2: Put "AdminLocal" LocalScript under "StarterPlayerScripts"

Step 3: Put the "Remotes" file in "ReplicatedStorage"

Step 4: Put the "Tools" file in "ReplicatedStorage"

If you don't want to use the Files mentioned you can change where these go but you have to change the script as well to account for the different path.
